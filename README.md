## How to compile the program
### Windows
1) install a c++ complier ie: Windows Subsystem for Linux (WSL)
2) open the terminal in cd/system
3) type $ make
4) type $ make run

### MacOS
1) open the terminal in cd/system
2) type $ make
3) type $ make run
(no installation required)

### Linux
1) open the terminal in cd/system
2) type $ make
3) type $ make run
(no installation required)
